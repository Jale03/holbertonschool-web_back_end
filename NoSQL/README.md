# NoSQL Project – MongoDB Basics

## Overview

This project introduces NoSQL databases using MongoDB. It focuses on basic database operations such as creating databases, inserting documents, querying data, and using Python with PyMongo.

## Learning Objectives

By the end of this project, I understand:

* What NoSQL means
* Difference between SQL and NoSQL
* What a document-based database is
* Basic MongoDB operations (CRUD)
* How to use MongoDB shell commands
* How to interact with MongoDB using Python (PyMongo)

## Requirements

* Ubuntu 20.04 LTS
* MongoDB (4.4+)
* Python 3.9 (for Python tasks)
* PyMongo 4.8.0

## Key Concepts

* MongoDB shell commands (show dbs, use, find, insert)
* Collections and documents
* JSON-like data structure
* Basic querying and filtering
* Python MongoDB driver (PyMongo)
