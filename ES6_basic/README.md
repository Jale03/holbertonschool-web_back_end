# ES6 Basics

## Description
This project covers the fundamentals of **ECMAScript 6 (ES6)**. It includes exercises on block-scoped variables, arrow functions, rest/spread parameters, and other modern JavaScript features.

## Learning Objectives
* Difference between `const` and `let`
* Arrow functions and default parameters
* Rest and spread parameters
* String templating
* Iterators and for-of loops
